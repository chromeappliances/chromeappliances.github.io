KDE Apps/Store

Included: Bunch of useful KDE Apps and a Linux App Store

Interested in a Linux App Store and some basic programming applications to make your life easier when using Linux on ChromeOS. Our KDE Appliance developed in partnership with Joe D. has you covered.

The KDE appliance comes with some KDE basics including Discover (an app store), Dolphin (file manager with SFTP), Konsole (terminal app with profiles for SSH connections), Okular (document viewer) and Kate (editor for text files and programming). You can easily extend and customize this appliance with Discover to browse and search for new applications, to install them, to run them, and to remove them if unwanted.
